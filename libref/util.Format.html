numeric_to_string(subject, p, f)

public static

Convert a numeric to a string.

Parameters :
p - The number of decimal places to produce (default 4).
f - A cset of flags. If c contains 'e' then the output is in scientific notation. 'E' is the same, but uses a a capital "E" in the result. If c contains ',' then commas are introduced into the non-fractional part of the number; if c contains '+' then a leading + is added to positive numbers. If c contains 's' then trailing fractional zeroes are replaced with spaces, whilst 'z' means drop such zeroes altogether. If c contains 'v' then any significant digits beyond SIG_DIGITS are output verbatim; otherwise these are replaced by zeroes, since they are usually meaningless.
