LB_line_clip(line, box)

Package :
ipl.glib
Type :
procedure

Note: Liang-Barsky algorithms (or variants) are used. If you have fast FP hardware, they are faster than Cohen-Sutherland (and *much* slower if you *don't*!). Anyway, they're more fun to code and easier to extend to 3-D.

LB_line_clip -- takes a 2-D line (two points) and returns it clipped to a box (normally the viewport).
