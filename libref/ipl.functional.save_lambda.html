save_lambda(e)

Package :
ipl.functional
Type :
procedure

Evaluates coexpression e, saving the value of _a and _l. This can be necessary if a lambda calls another lambda which exits via an exception or a runtime error.

Example :- 

    f := lambda{
       throw("Some problem")
    }

    g := lambda{{
       write("a[1]=",_a[1])
       save_lambda{try{ f(102) }}
       # Without save_lambda, _a[1] would now still be 102 (ie the arg given to f)
       write("a[1]=",_a[1])
    }}
