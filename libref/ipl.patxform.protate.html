protate(rows, dir)

Package :
ipl.patxform
Type :
procedure

Create rotated copy of a pattern. If dir is "cw" or "90", rotation is 90 degrees clockwise; if "ccw" or "-90", 90 degrees counter-clockwise. If dir is "180", rotation is 180 degrees. The default is "cw". rotate pattern
