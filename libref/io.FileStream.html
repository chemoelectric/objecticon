new(name, flags, perm, mode)

public

Open a new FileStream using the underlying system open function.
Parameters :
name - the file name
flags - the flags, selected from FileOpt, if perm and mode are omitted, defaults to FileOpt.RDONLY
perm - the create permissions to use if a file is created, selected from Mode.
mode - the resulting Stream's mode; if omitted this is deduced from flags parameter and the type of file being opened.

This method provides an interface to the system pread function.

NB - for consistency with seek and tell, offset is one based.

Parameters :
n - the maximum number of chars to read; n must be > 0.
offset - the one-based offset to seek to before reading.
