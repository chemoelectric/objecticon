add_string(s, pos)

public

Add a string at the given pos. If pos is not specified, the child is appended to the end.

This differs from the above add() method in that adjacent strings will be compacted together to form one longer string.

Parameters :
s - a string/ucs
pos - the pos to insert
Fails :
