callable(x, n)

Package :
lang
Type :
procedure

Try to convert x to something which can be invoked. This follows similar logic to that used in converting an expression for invocation (see the function general_call in invoke.r).

Parameters :
x - the thing to convert
n - the optional number of args for an operator. If null, then a valid operator name will be returned as a string; if 0 then no operator names are recognized; if 1, 2 or 3 then only an operator with that number of parameters will be recognized.
