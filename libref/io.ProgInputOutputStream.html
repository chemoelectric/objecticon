new(cmd, args, env, stderr)

public override

Overrides new() in io.SocketStream
Open a new readable and writeable Stream based on the given command. The first three parameters are the same as for the method posix.System.exec; stderr optionally gives the standard error of the command; by default the command inherits this from the parent. If the stderr param is the string "pipe", then stderr is sent to the pipe along with stdout.

For example :- 

    f := ProgInputOutputStream("myprog", []) | stop(&why)
    f.write("Some input")
    s := f.read()  # The response
    f.close()
