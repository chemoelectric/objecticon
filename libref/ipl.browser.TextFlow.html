fits_in_row_lr(w)

private

A special version of fits_in_row, used for determining if a left/right aligned component will fit in the row. The problem arises in a case like :-

FIRST<img src="j3.jpeg" align="right"/>SECOND

FIRST and SECOND form a blob, but row_width only includes FIRST after it has been output. Therefore fits_in_row may wrongly indicate that the img would fit in the row, rather than be deferred. The solution is to set row_blob to the end of the blob when FIRST is output, and use row_blob to see if the img would fit.
