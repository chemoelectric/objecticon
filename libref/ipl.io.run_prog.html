run_prog(prog, params, env)

Package :
ipl.io
Type :
procedure

A simple helper to run a program prog with the given optional parameter list and environment, and capture and return its output as a string. If the program exits with an error status code, then the procedure fails and sets &why appropriately.

Notes :-

  • This procedure only works on UNIX platforms.
  • stdin and stderr are both set to /dev/null.
  • The shell isn't invoked (unlike io.open(..., "pr")); so no shell syntax may be used.
  • The PATH is used to search for prog.

Example :- 
   s := run_prog("ls", ["-l", "/"])
