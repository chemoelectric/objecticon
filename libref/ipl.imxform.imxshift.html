imxshift(rows, shift, dir)

Package :
ipl.imxform
Type :
procedure

Create bit-shifted copy of an image matrix. If dir is "h", then the shift is horizontal; if "v", vertical. The default is horizontal. Positive shift is to the right for horizontal shifts, downward for vertical shifts. The default shift is 0 and the default direction is horizontal.
