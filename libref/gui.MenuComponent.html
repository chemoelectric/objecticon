This is the superclass of all the objects that make up the menu system (other than the top level Components of course).

It encapsulates the position and space of labels in the parent menu, as well as methods which allow this item to be a popup submenu. A submenu need not necessarily contain other MenuComponents; see for example the Palette class. A conventional submenu, which does contain children, is in the Menu class.
