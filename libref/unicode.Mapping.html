fold_chars(s)

public static

Return a co-expression, the results of which are the individual characters that would comprise string(fold(s)). For example, 
    fold_chars(u"Spaß")
generates 5 characters, namely "s", "p", "a", "s", "s".

This function is useful for doing comparisons. For example, the ipl.sort.convert_cmp and ipl.sort.coex_cmp procedures can be used to create a comparator function as follows :- 

    import ipl.sort
    ...
    f := convert_cmp(Mapping.fold_chars, coex_cmp())

This comparator could then be used to compare strings directly or perhaps with a SortTable :- 

    import datastruct
    ...
    f(u"Spaß", u"SPASS")   # Gives 0 (ie, equal).
    ...
    t := SortTable(, f)
    t.insert(u"Gefäß", 99)
    t.member(u"GEFÄSS")    # 99
    t.member(u"GEFÄß")     # also 99
    t.select(1)            # u"Gefäß"
