UndoManager

Type :
class
Superclasses :
undo.CompoundEdit
Package :

UndoManager is a CompoundEdit. Until it is closed it allows undos and redos within its list of edits, moving a pointer into the list appropriately.

Once closed, it behaves just like a closed CompoundEdit.
