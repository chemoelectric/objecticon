exec(cmd, args, env)

public static

This provides a slightly easier to use interface to execve. The first argument is searched on the PATH and if found pushed onto the arg list (making the arg0 of the target program). Then execve is called. For example :- 
   System.exec("ls", ["-l", "/home"])
Parameters :
cmd - the name of the command
args - the arg list, excluding the command name. Defaults to []
env - the env list, defaults to the current environment if omitted

execve(f, args, env)

public static native

Replace the executing Icon program with a new program, using the execve system call. The first parameter is the file to execute, the second parameter is a list of strings which must have at least one parameter, and the third optional parameter is also a list of strings, which provide the environment variables - if omitted then the current program's environment is passed (obtained via the environ variable). For example :- 
         System.execve("/bin/ls", ["/bin/ls", "-l", "/home"],
                                  ["ABC=DEF", "NUM=123"])
