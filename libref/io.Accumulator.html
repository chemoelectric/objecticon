Accumulator

Superclasses :

Package :

This is a write-only Stream which can be used to accumulate strings in a list, which can then be concatenated together in an efficient loop (see str()). RamStream can do the same thing (and more), but this class has the advantage that it doesn't need to be closed, since it uses no external resources (close() is a no-op).

For example :- 

  f := Accumulator()
  f.out("Hello ")
  f.write("World")
  s := f.str()     # "Hello World\n"
  # No need to close() f
