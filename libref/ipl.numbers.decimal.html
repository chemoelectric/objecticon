decimal(i, j)

Package :
ipl.numbers
Type :
procedure

decimal expansion of i / j; terminates when expansion terminates or the end of a recurring period is reached. The format of the returned value is <integer>.<seq>, where <seq> is a string a decimal digits if the expansion is finite but <pre>[<recurr>] if it is not, where <pre> is a string of decimal digits (possibly empty) before the recurring part.
