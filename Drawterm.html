Introduction

This is a version of drawterm, with enhancements and modifications to the X11 interface, as described below.

Scaling

The X11 window is a scaled view of the Plan 9 screen. The default scaling factor used assumes the Plan 9 screen is 96 dpi, and calculates the dpi of the X11 desktop using the various X11 screen size functions. The scale factor may be manually set using the -S option.

A scale factor of 1.0 means a 1:1 mapping of Plan 9 to X, 2.0 means each Plan 9 pixel is doubled in size, and so on.

Resizing the X11 window changes the scaling factor, but not the size of the underlying Plan 9 screen.

Full-screen mode

Pressing F12 toggles full-screen mode. The key may be configured with the -K option.

Clipboard

Writing to and reading from the snarf file copies to and from the X11 clipboard selection. The primary selection is not used.

Command line options

The following X11-specific options are available :-

  • -A Use the X11 arrow cursor rather than the Plan 9 arrow cursor as the default cursor.
  • -F Start in full-screen mode.
  • -K KEY Set the key to use to toggle full-screen mode; KEY should be recognized by X11â€™s XStringToKeysym function. The default key is F12.
  • -M Suppress the full-screen exit message.
  • -R WxH Set the size in unscaled Plan 9 pixels of the window; the visible window will be scaled to a different size.
  • -S VAL Set the scaling factor to use.
  • -E Center the Plan 9 screen within the window, rather than aligning it with the top-left.
  • -B VAL Set the background colour to use to fill margins around the Plan 9 screen. VAL should be a hex integer. The default is black.

There is also an additional generic option, -p can be used to set the login password.

Dependencies

In addition to the X11 client libraries, this version needs the cairo and XRender client libaries and headers. On a ubuntu system, installing the following packages will provide these :-

libx11-dev
libxrender-dev
libcairo2-dev

Compilation

Just run make in the drawterm directory. Hopefully an executable drawterm in that directory will be the result.
