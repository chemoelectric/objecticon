replacec(s, c, t)

Package :
ipl.strings
Type :
procedure

Replace each occurence in string s of a character in cset c by the string t (which may be of any length). For example :- 

    replacec("The sum is $2000.00", &digits, "*")
    gives "The sum is $****.**"

If t is the empty string, then this is equivalent to deletec.

The result type is the type of s.
