lpb(x, op)

Package :
ipl.functional
Type :
procedure

Convenient function to get the binary operator op, partially applied with x as the left argument. For example 

   lpb(2, "*")
returns a function which doubles its argument, whilst 
   lpb(0, "<")
returns a function which succeeds and returns its argument if it is greater than 0.
