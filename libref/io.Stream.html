read_exactly(n)

public

Read n chars from the stream. This method will try and return exactly n characters, and will only return less because of reaching the end-of-file.

If end-of-file is encountered before any characters were read, then &null is returned.

read_line()

public

Read a complete line from the stream, reading up to the next "\n" or end-of-file. In the former case, the "\n" and also any "\r" immediately before it is stripped from the result.
