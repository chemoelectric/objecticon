be_unsigned_string(i)

Package :
ipl.bincvt
Type :
procedure

Returns a string which is a binary representation of non-negative integer i, in the fewest number of characters necessary, so that for any valid i, 

    be_unsigned(be_unsigned_string(i)) = i
The output may be shorter than be_signed_string; for example 
 be_signed_string(200) == "\x00\xc8"
 be_unsigned_string(200) == "\xc8"
Obviously the output of this procedure shouldn't be given to be_signed (the above example of 200 would give a negative result).
