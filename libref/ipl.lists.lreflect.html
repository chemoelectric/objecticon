lreflect(L, i)

Package :
ipl.lists
Type :
procedure

List reflection. Returns L concatenated with its reversal to produce palindrome; the values of i determine "end conditions" for the reversal:

  • 0 omit first and last elements; default
  • 1 omit first element
  • 2 omit last element
  • 3 don't omit element
