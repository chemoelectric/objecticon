DrawGroove(win, x1, y1, x2, y2, w) -- draw a grooved line

If w > 0, draw groove of width 2. If w = 0, erase groove/ridge of width 2. If w < 0, draw ridge of width 2.

Horizontal and vertical grooves fill the same pixels as lines drawn line_width=2. Angled grooves are not necessarily the same, though. draw grooved line
