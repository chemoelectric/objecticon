save_var(e, var, i)

Package :
util
Type :
procedure

Invoke co-expression e, generating its results, in such a way that changes to var made by e will not be seen by the caller, and vice versa.

Parameters :
e - a co-expression to evaluate
var - a co-expression generating a global variable.
i - an optional initial value for the global variable within e

Example :- 
  # Cleanup some io stuff, without affecting global vars g.
  save_var{ io_cleanup(), g }
