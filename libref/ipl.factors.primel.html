primel()

Package :
ipl.factors
Type :
procedure

Generates the primes from a precompiled list, "primes.lst", in the directory indicated by the environment variable DPATH (see dopen). If the sequence won't be generated to exhaustion, enclose the call in util.clean to ensure the underlying file is closed; eg 

  clean{ every write(primel() \ 20) }
