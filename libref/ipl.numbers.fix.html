fix(i, j, w, d)

Package :
ipl.numbers
Type :
procedure

formats i / j as a real (floating-point) number in a field of width w with d digits to the right of the decimal point, if possible. j defaults to 1, w to 8, and d to 3. If w is less than 3 it is set to 3. If d is less than 1, it is set to 1. The function fails if j is 0 or if the number cannot be formatted.
