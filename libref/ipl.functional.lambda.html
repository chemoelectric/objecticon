lambda(e)

Package :
ipl.functional
Type :
procedure

Given a co-expression e, this returns a function whose results sequence is generated from a refreshed copy of e. Arguments to the function are passed to e via a global variable _a. For example: 

  f := lambda{ _a[1] + 100 }

sets f to be a function returning its first argument plus 100.

To simulate local variables, another global, _l, can be used. It can be set to any value.

Other examples:- 

  # Return the sum of the arguments, eg f(1,2,3) -> 6
  # - note that _l must be dereferenced, otherwise the caller sees the
  #   value before the function is called.
  f := lambda{{ _l := 0; every _l +:= !_a ; ._l }}

An alternative to this procedure is argf, which passes arguments by way of an activation.
