new(notty, f)

public override

Overrides new() in io.LineBasedInputStream
Create a LineEditorStream.

Parameters :
notty - a flag, implicitly set on non-Unix platforms. If set, then the line editing facilities are disabled

f - a Stream to read from if line editing is disabled. If omitted, FileStream.stdin is used.

Examples :- 
   # On Unix use line editing, on other platforms use stdin
   s := LineEditorStream()

   # Don't use line editing; read from the default stdin.
   s := LineEditorStream(&yes)

   # Don't use line editing; read from the given stream.
   s := LineEditorStream(&yes, 
                         TaskStream(FileStream.stdin, task))
