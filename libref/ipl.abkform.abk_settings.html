HP 95LX Appointment Book Settings Record:

Byte Offset Name Type Contents

0 StartTime int Daily display start time as the number of minutes past midnight. 2 Granularity int Daily display time line granularity in minutes. 4 AlarmEnable char 1 = on, 0 = off 5 LeadTime char Alarm default lead time in minutes. 6 CarryForward char To do carry forward default, 1 = on, 0 = off.
