Variable detail

convert

public const

The conversion procedure; this should be one of the built-in conversion functions (integer, string, etc), util.flag, or another procedure which converts the string parameter, and sets &why on failure. If this field is null, then the option takes no value.

Source - options.icn

help

public const

The descriptive help string produced when "-?" is used. This may be split into two parts, separated by a # character; the part to the left is the string used to describe the option parameter in the help output. For example :- 
   Opt("f", string, "FILE#Load values from FILE")
Would produce a help string :- 
   -f FILE             Load values from FILE
