This class extends LineBasedScrollArea to provide selection on lines, event handling and selection handling.

Mouse and key movements are briefly as follows.

  • Mouse movements.
    • middle and right mouse just move the cursor and don't select
    • left mouse moves the cursor and selects. In select many mode :-
      • shift modifier selects all from old cursor to new
      • ctrl modifier toggles one row on or off

  • Key movements. In selection on key moves mode :-
    • control modifier means just move, no selection.
    • shift modifier means (in select many mode) select all from old cursor to new.

  • Return key.
    • In select many mode, shift or control modifier means toggle the cursor row's selection.
    • Otherwise, the cursor row alone is selected.

  • Escape key clears the selection.

  • Control-A selects all rows (in select many mode).

draw_line(xp, yp, lh, i, sel, cur, hi, ov)

public override

Overrides draw_line() in gui.LineBasedScrollArea
This method is overridden by the subclass to draw the given line at the given position, as well as the selection drawing. Alternatively, the simpler method draw_line_data() below can be overridden - this just has to draw the data, and leaves the selection drawing to be done by this class.

Parameters :
xp - The x position it should be drawn at
yp - The y position it should be drawn at
lh - The line height allowed for this line
i - The line number to draw
sel - If non-null, the line must be drawn as selected
cur - Indicates the cursor for this row; &null indicates none, "f" indicates focus cursor "d" indicates disabled cursor.
hi - Indicates the dnd highlight for this row; &null indicates none, "t" indicates top highlight, "m" indicates middle highlight, "b" indicates bottom highlight.
ov - If non-null, the line must be drawn as showing the cursor over the line
