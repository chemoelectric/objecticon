A ABorder (so named to avoid a clash with the Border component), is an object which represents a border. It reports the widths of the four edges of its border. Given a rectangle, it also draws the border on a Window.

An ABorder also subclasses XYAlign. This lets it position a Size within a Rect by "floating" it within the bordered inner area according to the alignment.
