backto(c, s, i, j)

Package :
ipl.scan
Type :
procedure

Search backwards in s[i:j], for chars in cset c, beginning at the end. For each one found, the position just to the right is suspended.

For example - 

   backto('aeiou', "quick brown fox") -> {15,10,4,3}
Note that, if no parameters are defaulted, 
   backto(c, s, i, j) - 1
gives the same sequence of results as 
   upto(c, s, i, j)
but in reverse order.

Defaults:

  • s - &subject
  • i - &pos if s is defaulted, otherwise 0
  • j - 1
