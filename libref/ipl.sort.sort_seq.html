sort_seq(el, cmp)

Package :
ipl.sort
Type :
procedure

This procedure takes a list of co-expressions and a comparator function, and generates the results of the co-expressions in order, according to the comparator. Each co-expression must generate its results in the order indicated by the comparator.

For example :- 

 sort_seq( [ create 5 to 20 by 5,
             create 11 | 200,
             create 11 | 99 ] )
generates :- 
         5  10  11  11  15  20  99  200

Parameters :
el - the list of co-expressions
cmp - a comparator function, default Internal.compare
