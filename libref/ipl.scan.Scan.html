q(e)

public

This method allows other string scanning functions to be used with this class. The co-expression e is evaluated for its results, with &subject and &pos being repeatedly swapped with ksubject and kpos. For example :- 

   s := Scan("Mary had a little lamb").set_pos(0)
   s.q{ backto('a') }     # Generates 21, 11, 8, 3.
