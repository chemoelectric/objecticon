A Stream which takes its data from a co-expression. Each result provides a chunk of output of arbitrary size. End-of-file is signalled by the co-expression generating &null and errors are signalled by it failing and setting &why.

For example :- 

    # A stream containing a string repeated indefinitely.
    e := GeneratorStream{ |"The quick brown fox" }

    # A stream containing lines of random digits.
    e := GeneratorStream{ |?(&digits ++ '\n') }
