use(e1, e2, e3)

Package :
util
Type :
procedure

This procedure is used to automatically close a Stream or other resource. The first expression e1 produces the resource, the second e2 uses it, and the third expression e3 then closes it. The result of e2 is returned, if all three expressions succeed, otherwise the procedure fails.

Each co-expression is optional, but if e1 is omitted e3 must be provided; otherwise e3 defaults to calling close on the resource.

Example :- 

    s := use { f := open("some.txt"),
               do_something_with(f) }

See also clean() below.
