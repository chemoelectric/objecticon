altlists(l, alts)

Package :
ipl.lists
Type :
procedure

Given a list l and a table (int->list) of alternative elements, generate all possible modified lists. For example, given l=[1,2,3] and t={1->a,b 3->c,d} we generate :-

[a,2,c],[a,2,d],[a,2,3],[b,2,c],[b,2,d],[b,2,3],[1,2,c],[1,2,d]

Note that the original list l is not generated, only modifications. The same list is returned each time.

The no. of results generated is :- (product of (*!alts + 1)) - 1 For the example above, (2+1)*(2+1) - 1 = 8.
