factorizations(n)

Package :
ipl.factors
Type :
procedure

Generate a sequence of lists giving all the possible factorizations of positive integer n. For example, if n is 24, then :- 

  [24], [2,12], [4,6], [2,2,6]
  [3,8], [2,3,4] and [2,2,2,3]
is generated.

Notes :-

  • 1 is never included in any result.
  • the factors in each list are in ascending order,
  • the results are in no special order, although the first result always has one element and the last all the prime factors.
  • no list is repeated,
  • each list is a distinct list instance,
  • factorizations(1) returns the empty list.
