mergesort(l, cmp, pos1, pos2)

Package :
ipl.sort
Type :
procedure

A merge sort implementation. Unlike qsort, this returns a sorted copy, leaving the original unchanged. If pos1 and pos2 are given, then only the sublist l[pos1:pos2] is sorted, and that sublist is returned.

Parameters :
l - the list to sort
cmp - a comparator function, default Internal.compare
pos1 - the first slice index, default 1
pos2 - the second slice index, default 0
