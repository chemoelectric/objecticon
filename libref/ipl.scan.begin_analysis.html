begin_analysis(s, i, j)

Package :
ipl.scan
Type :
procedure

This procedure can be used to normalize the three parameters commonly given to string analysis functions (like upto, find etc).

The defaults applied are :-

  • s - &subject
  • i - &pos if s is defaulted, otherwise 1
  • j - 0

The result is a StringAnalysis record with the normalized values of s, i and j, or failure if i and/or j are out of bounds.

Note that the returned values in the StringAnalysis record will be adjusted so that i <= j.
