get_row()

public

This uses fetch_row() and fetch_fields() together to provide a rather more intelligent version of fetch_row(). The column types are examined and the strings are converted to appropriate Icon objects: integer for the integer types, real for the floating point types, Time for the date/time types, and string for the char/blob types.
