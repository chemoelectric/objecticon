Prog

Type :
final abstract class
Package :
lang

In the methods below, the term "program" refers to the main co-expression of either the root program, or a program loaded via Prog.load below. The parameters named P are always programs and default to the currently executing program if omitted.

load(s, arglist, blocksize, stringsize)

public static native

Load and return a program from file s.

Parameters :
s - the file name of the program.
arglist - an optional list of arguments to pass to the program's main procedure; they need not be strings.
blocksize - the optional size of the program's block region. If omitted, then the top-level program's block region size is used; if zero then the program shares its loader's block region.
stringsize - the size of the string region, with similar defaults to blocksize.
