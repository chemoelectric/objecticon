RandomFromGenerator

Type :
class
Superclasses :
ipl.random.RandomWithModulus
Package :
ipl.random

This class is a RandomWithModulus in which a co-expression provides the values returned by get(). This enables results from other generators to be easily modified or combined.

For convenience, the constructor optionally accepts PDCO-style parameters.

For example :- 

   import ipl.pdco
   ...
   # A generator which uses every third result from a PCG32 source.
   r1 := PCG32()
   r := RandomFromGenerator{ r1.m, Index{ r1.gen(), seq(3, 3) } }

   # A generator which XORs two LCGs' results.
   r1 := LCG(53, 1, 2^16) ; r2 := LCG(525, 11, 2^16)
   r := RandomFromGenerator{ r1.m, ixor!Parallel{ r1.gen(), r2.gen() } }
