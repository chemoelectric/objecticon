An abstract class for random generators which generate integers in the range of an arbitrary fixed modulus (usually a power of 2).

The modulus m is supplied in the constructor, and a single method, get() is implemented which generates numbers in the range [0, m).

get_n(p)

public

This method combines p random numbers from get() to give a result in the range [0, m^p). For example, if m=2^5, get_n(2) would return a number in the range [0, m^2) = [0, 2^10).
