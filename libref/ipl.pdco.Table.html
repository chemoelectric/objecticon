Table(e, def)

Package :
ipl.pdco
Type :
procedure

Construct a table from a result sequence. The expression e should produce 2-element lists (or records) representing key-value pairs to add to the table.

An optional second parameter provides a default value for the table.

For example :- 

   Table{(i := 1 to 4, [i, i * i])}
produces the table 
   {1->1, 2->4, 3->9, 4->16}
