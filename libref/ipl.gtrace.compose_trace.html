compose_trace(trace, call_1, call_2)

Package :
ipl.gtrace
Type :
procedure

compose_trace(call_1, call_2) composes the trace for call_1 with the trace for call_2; that is, the trace for call_1 is passed through call_2. For example, if call_1 traces a circle and call_2 draws a star, the result is a star on each point of the circle.

The procedure assumes that the first two arguments to call_2 are the x and y coordinates of the point in which it is interested (standard trace format).
