RecordingSurface

Type :
final class
Superclasses :
cairo.Surface
Package :
cairo

This class wraps a cairo recording surface, which can be used to record all drawing operations at the highest level (that is, the level of paint, mask, stroke, fill and show_text_glyphs). The recording surface can then be "replayed" against any target surface by using it as a source to drawing operations.
