divisions(x)

Package :
ipl.lists
Type :
procedure

Given a list or set x, generate all possible divisions into sublists (or subsets). For example, if x is [1,2,3] we generate :- 

[ [1,2,3] ]
[ [1], [2,3] ]
[ [1,2], [3] ]
[ [2], [1,3] ]
[ [1], [2], [3] ]
Notes :-

  • the empty list or set is never included in any result (unless x is empty).
  • for lists, the order of elements within each sublist is maintained.
  • the results are in no special order, although the first result always has a single element (a copy of x) and the last has *x elements.
  • each result list and the sublists or subsets are distinct instances.
