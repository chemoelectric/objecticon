PAL_Order(p) -- return reordered palette chars (internal routine)

Normal order for color cube is sorted r/g/b, then extra grays. Reorder by g/r/b followed by full set of grays, including duplicates, back to black. Returns unmodified list of characters for c1 and grayscale palettes.
