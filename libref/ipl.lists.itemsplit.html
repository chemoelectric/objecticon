itemsplit(x)

Package :
ipl.lists
Type :
procedure

Generate the elements of x (a list or a set), together with the other elements of x. So for [1,2,3] we generate 1,[2,3]; 2,[1,3]; and 3,[1,2].

An itemsplit_rec record is generated with each result; note that the same record instance and contained structure is suspended each time.
