String(e, sep)

Package :
ipl.pdco
Type :
procedure

Build a string from the results of the co-expression e, separated by the string sep (default ""). If sep is ucs, or any of the results of e are ucs, then the result is ucs, otherwise it is a string.

Note that for some use cases, String2 below may be much faster than this procedure.
