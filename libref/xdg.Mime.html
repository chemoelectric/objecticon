Generate the alternative mime types for the given mime type. This traverses the graph formed from the subclasses and aliases files, in a breadth-first fashion, and generates all possible alternatives.

gen_all_icons(mt)

public static

Generate all the icons defined for the given mime type, namely any mappings from the "icons" files, then the default icon formed from substituting "/" for "-", and then any mappings from the "generic-icons" files. This method will always generate at least one result.

public static
