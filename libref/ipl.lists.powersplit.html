powersplit(x)

Package :
ipl.lists
Type :
procedure

Generate all possible 2^(*x) splits into two parts of x (a list or a set). For example, given x=[1,2] we generate four pairs :- [1,2],[]; [1],[2]; [2],[1]; and [],[1,2].

A powersplit_rec record is generated with each result; note that the same record instance and contained structures is suspended each time, although these are distinct from x.
