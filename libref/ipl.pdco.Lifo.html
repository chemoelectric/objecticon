Lifo(L[])

Package :
ipl.pdco
Type :
procedure

Generate the results from the several expressions in the same left-to-right order as in normal icon evaluation.

For example Lifo{1 to 3, !"ab"} generates [1,"a"], [1,"b"], [2, "a"] etc.

Note that the same list instance is suspended each time, and should not be altered.
