turtle(win, pos, direction, color)

Package :

A simple implementation of 'turtle' graphics for multiple windows one can have more than one turtle simultaneously active In a turtle, the color field (if used) must be a co-expressions that produces the color. This allows the turtle to change color as it runs. In the simplest case, construct the turtle with a co-expression the repeatedly supplies the the same color: create |"red" ###########################################################################
