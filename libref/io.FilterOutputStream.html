FilterOutputStream

Type :
class
Superclasses :
io.FilterStreamBase
Package :
io

This stream filters output written to it through a background process and outputs the results to an arbitrary sink Stream.

For example :- 

  # Encode s, putting the result in a RamStream
  r := RamStream()
  g := FilterOutputStream(r, "gzip", ["-c"])
  g.write(s)

Overrides close() in util.HasClose
Close the stream. This waits for the child process to terminate, flushing any remaining output to sink. NB: sink and err_sink are not closed unless the set_close_underlying has been set to &yes.
