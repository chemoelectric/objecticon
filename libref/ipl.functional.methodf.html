methodf(field)

Package :
ipl.functional
Type :
procedure

Returns a function that takes its first argument, obtains the given field from it, and then applies the remaining arguments to the resulting method.

Example :- 

  f := methodf("abc")
  f(obj, 1, 2, 3)   # invokes obj.abc(1, 2, 3)
