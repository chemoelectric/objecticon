index_cmp(i, f)

Package :
ipl.sort
Type :
procedure

Return a compare function which applies the comparator function f to the ith field of each item. f defaults to Internal.compare.

For example :- 

    l := [ [1,54,2], [4,56,23], [4325,1,23], [123,4576,23]]
    # sort by the middle element of each item
    qsort(l, index_cmp(2))
