cmp(x, f, y)

Package :
numbers
Type :
procedure

Compare x and y using the binary operator f. The two values are first converted and made the same type by align_real_numbers().

For example :- 

     x := Dec("0.4") ; y := Rat("1/3")
     if cmp(x, ">", y) then write("x is greater than y")

Parameters :
x - something convertible to Dec or Rat
f - something convertible to a comparison operator.
y - something convertible to Dec or Rat
