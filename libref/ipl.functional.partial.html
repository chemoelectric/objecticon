partial(f, spec, l[])

Package :
ipl.functional
Type :
procedure

Returns a function which modifies its arguments according to spec, before applying them to f. The modification is specified by the following characters in spec:- "v": add the next value from the arguments which follow the spec parameter, or &null if none remain. "V": add the remaining values. "a": add the next argument, or &null if none remain. "A": add the remaining arguments. "-": skip the next argument.

Example :- 

    f := partial(write, "vvvAV", "a", "b", "c", "p", "q", "r")
    f("x","y","z")   # writes abcxyzpqr
