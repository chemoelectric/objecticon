re_encode(s, c1, c2)

public static

Modify the percent encoding in the given string s.

For currently unencoded characters, those in c1 are left as-is, others are percent-encoded.

For currently encoded characters, those in c2 are decoded, and others are left percent-encoded.

c1 defaults to UNRESERVED, and c2 defaults to c1.

To work properly, c1 should not include %. For consistency (meaning a further application would produce the same result), c2 should be a subset of c1.

Example :- turn a URL with utf-8 characters into a string for display, showing the utf-8 characters unescaped :- 

    URL.re_encode(url.str(), ~'%', ~&ascii)
