begin_rev_analysis(s, i, j)

Package :
ipl.scan
Type :
procedure

This procedure is similar to begin_analysis, but provides different defaults, more suited to procedures that search backwards, as follows :-

  • s - &subject
  • i - &pos if s is defaulted, otherwise 0
  • j - 1

Note that the returned values in a StringAnalysis record will still be adjusted so that i <= j.
