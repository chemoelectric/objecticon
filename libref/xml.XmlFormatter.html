A formatter for XML documents. The default formatter behaviour is to output the string content as held in the children array (this may or may not have had whitespace removed depending on whether validation against a DTD occurred), and without indentation.

set_as_read(s)

public

Configure the formatter so that the string content will be output as read from the input; ie the formatter uses the whitespace_children list rather than the children list. The default is off.

set_indent(n)

public

Ensure that each opening element appears with an indent of n chars. Whitespace will be inserted as appropriate. By default, no indentation is done.
