StringStream

Type :
class
Superclasses :
io.Stream
Package :
io

This is a stream based on an Icon string. An initial string can be provided to the constructor. Reading from the stream will be quite efficient, but writing will be relatively slow, particularly if the string is long. For a much faster alternative, see RamStream.
