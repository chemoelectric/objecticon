be_string(i, size)

Package :
ipl.bincvt
Type :
procedure

Returns a string from the low order size bytes of integer i, assuming big-endian ordering. If i is negative, then a two's complement representation is the result. For example :- 

 be_string(1234, 4) == "\x00\x00\x04\xd2"
 be_string(-1234, 4) == "\xff\xff\xfb."
For all i in the range 0 <= i <= 2^(8*size) - 1, 
 be_unsigned(be_string(i, size)) = i
and in the range -(2^((8*size)-1)) <= i <= 2^((8*size)-1) - 1 
 be_signed(be_string(i, size)) = i
