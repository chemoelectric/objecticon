calculate_line_height()

public abstract

This method is overridden by the subclass to work out the line height. It should return either an integer, indicating that each line is the same height, or a list, giving the height of each line.

Because this operation is potentially expensive, it is called as late as possible and its value cached.

get_line_info(r)

public

Get line info for the rectangle r, which defaults to the view. If given, r must be within the view. The result gives the first and last lines in r and the first and last wholly visible lines in r. For the latter, if there are no such lines then the first and last lines are returned. Fails if there are no data lines in r.
