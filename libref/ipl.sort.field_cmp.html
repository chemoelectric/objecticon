field_cmp(q, f)

Package :
ipl.sort
Type :
procedure

Return a compare function which applies the comparator function f to the field named q in each item. f defaults to Internal.compare.

For example :- 

    l := [Point(1,2),Point(34,4),Point(3,56),Point(23,55)]
    # sort by the x element of each item
    qsort(l, field_cmp("x"))
