lambda1(e)

Package :
ipl.functional
Type :
procedure

This is just like lambda, but the global variable _a is assigned to the first argument, and other arguments are discarded. This makes functions which take a single argument rather more compact. For example :- 

  # A factorial function 
  g := lambda1{ if _a = 0 then 1 else _a * g(_a - 1) }
