extended_euclid(u, v)

Package :
ipl.numbers
Type :
procedure

Calculate the gcd and coefficients of BÃ©zout's identity for positive integers u and v, using the extended Euclidean algorithm, adapted from Knuth vol.2, 2nd ed., p325.

Returns a record r with fields u1, v1, u2, v2, gcd, giving the two minimal pairs of coefficients :- 

    u * r.u1 + v * r.v1 = r.gcd, r.u1 > 0, r.v1 <= 0
    u * r.u2 + v * r.v2 = r.gcd, r.u2 <= 0, r.v2 > 0
