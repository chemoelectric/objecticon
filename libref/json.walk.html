walk(v, a[])

Package :
json
Type :
procedure

A helpful function for descending through a structure (v) of lists and tables returned by json parsing. The list a gives the item to select at each level. Lists must match integer indices and tables must have the exact key. For example :- 

   # Typical parse output
   v := [9,120,8984,24, table(,"x",2,"y",4)]
   walk(v, 5,"x")        # Gives 2

If the list of selectors doesn't lead to an element, then the procedure fails, setting &why to indicate where the search ended.
