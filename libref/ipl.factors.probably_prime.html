probably_prime(n, c)

Package :
ipl.factors
Type :
procedure

Probabilistic prime testing. Succeeds if n is "probably" prime. Adapted from Knuth vol. 2, 2nd ed. sec 4.5.4.

c is the confidence level (default 50). The algorithm will give a false positive (ie say a number is prime when it isn't) with probability about (1/4)^c (for c=50 this is about 10^-30).
