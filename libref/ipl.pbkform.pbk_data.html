HP 95LX Phone Book Data Record:

Byte Offset Name Type Contents

0 RecordType char 1 (01h) 1 RecordLength int Number of bytes in remainder of this data record, see note below. 3 NameLength char Length of name text in bytes. 4 NumberLength char Length on number text in bytes. 5 AddressLength int Length of address text in bytes. 7 NameText ASCII Name text, 30 characters maximum. 7+NameLength NumberText ASCII Number text, 30 characters maximum. 7+NameLength+ NumberLength AddressText ASCII Address text where the null character is used as the line terminator. Addresses are limited to a maximum of 8 lines of 39 characters per line (not counting the line terminator).
