imxdouble(rows, dir)

Package :
ipl.imxform
Type :
procedure

Creates a tile in which each pixel doubled. dir determines the direction in which the doubling is done. If dir is "b" or null, it's done both horizontally and vertically. If dir is "v", it's only done vertically, while if dir is "h", it's done only horizontally.
