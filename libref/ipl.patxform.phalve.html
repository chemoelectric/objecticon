phalve(rows, dir, choice)

Package :
ipl.patxform
Type :
procedure

Creates a tile in every other pixel is discarded. dir determines the direction is which the halving is done. If dir is "b" or null, it's done both vertically and horizontally. If dir is "v", it's only done vertically, while if dir is "v", it's done only vertically. If choice is "o" or null, odd-numbered rows or columns are kept; if "e", the even-numbered ones. halve pattern by bits
