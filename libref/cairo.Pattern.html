This class wraps a cairo pattern. There can be several instances wrapping the same cairo object. Each one has a reference to the underlying object and must be closed separately.

to_pattern(v)

public static

This is a convenient method to create a Pattern instance from a variety of objects. v can be a Surface, a Window, a Pixels, raw image data, or a filename of an image.

Note that if a Pixels is given, then a copy is made, and future changes to the Pixels have no effect on the Pattern.
