This is a common superclass for a set-like data structure.

A subclass must implement different optional methods depending on its capability mode, as follows :-

  • ORDER - back(), rank() and select()
  • INSERT - insert()
  • DELETE - delete()

There are several other abstract methods which all classes must of course implement, and others which may be overridden to provide more efficient implementations than the default.

Note that the behaviour of the various generator methods should the set be concurrently modified during the course of generation, depends upon the particular set implementation.
